DESCRIPCIÓN
Encontrarás varios directorios, que te explicamos a continuación:

content/
Acá encontrarás archivos de contenido en formato HTML.

data/
Acá encontrarás archivos de datos en formato JSON.

- users.json: contiene una lista de datos de usuario.
- courses.json: contiene una lista de datos de curso.
- enrollments: contiene una lista de matriculaciones, aquí se relacionan los usuarios con los cursos.
- pages: contiene una lista de datos de páginas de contenido, aquí se relacionan los contenidos con los cursos.

media/
Acá encontrarás archivos como imágenes y videos.

templates/
Acá encontrarás archivos HTML a manera de plantillas.

PRUEBA
La prueba consiste en implementar una aplicación web con tres vistas:

1. Login
Una vista de login, con los campos de usuario, contraseña y un botón.

2. Home
Una vista con la información personal del usuario y los cursos a los que está inscrito.

3. Contenido
Una vista con el contenido de la primera página del primer curso (A1).

OBSERVACIONES
- La validación de usuario y contraseña puede ser simulada.
- Ten en cuenta el diseño responsivo para móviles.

TECNOLOGÍAS
Puedes elegir los frameworks que desees para front y back. El back debe utilizar NodeJS.

ENTREGABLE
Debes entregar a vuelta de correo un archivo zip con tu código y una descripción en texto de la instalación, uso y demás aspectos que quieras comentar.
